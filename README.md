# About

Templates of scenarios using Appium.

Appium is E2E test automation tool
- http://appium.io/

# Requirement
appium@1.3.x

# Setup

```
$ ./script/setup.sh
```

# Build aplication

```
$ ./script/xcodebuild.sh
```

# Run test

```
$ ./run_test.sh staging
```

# capabilities
```config/ios_capabilities.rb``` or ```config/android_capabilities.rb```
