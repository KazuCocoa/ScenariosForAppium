# About

Templates of scenarios using Appium.

Appium is E2E test automation tool
- http://appium.io/

# Requirement
appium@1.2.x

# Caution
## Run Appium
Start Appium

```
$ appium
```

## Run Test

```
$ ./run_test.sh staging
```

You should set modify target application module from ```config/ios_capabilities.rb``` or ```config/android_capabilities.rb```
